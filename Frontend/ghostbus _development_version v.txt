➤ 8. SELLER DASHBOARD & UPLOAD PIPELINE

Sidebar

* Dashboard
* Upload
* Tracks (Status: Live, Pending, Rejected, Sold)
* Earnings
* Messages (Gig System)

➤➤ Upload Flow (STRICT 5-STEP PIPELINE)

Step 1: Metadata

* Title
* Genre
* Sub-genre
* BPM
* Key
* Description

Step 2: Files

* Upload Mastered WAV
* Upload Unmastered WAV
* Upload Stems (ZIP)
* Upload MIDI (ZIP)
* Upload Artwork

Step 3: Transparency

* Required toggle:

  * 100% Original
  * Contains Royalty-Free Loops

Step 4: Pricing

* Seller inputs price
* System enforces minimum and maximum limits based on seller tier

Step 5: Automated Verification & A&R Lock (CRITICAL)

File Validation:

* Backend checks ZIP size and structure
* Prevent empty or low-quality stem uploads

Uniqueness Scan:

* System sends Mastered WAV to MRT API (e.g., DJMonitor)
* If match found → instant REJECT

A&R Lock:

* If passed → status = PENDING_REVIEW
* Track is NOT published automatically
* Sent to admin moderation queue for manual approval

➤ 12. PAYMENT SYSTEM & TIERED COMMISSIONS (STRICT)

➤➤ Payment Infrastructure

* Stripe ONLY
* Stripe Connect for payouts
* No manual payout processing

➤➤ Commission Logic (Dynamic System)

Tier 1 (New Seller): Platform fee = 36%
Tier 2 (Mid-Volume): Platform fee = 30%
Tier 3 (High-Volume): Platform fee = 25%

* Backend calculates commission dynamically based on seller lifetime sales

➤➤ Stripe Connect Flow

* Remaining balance routed instantly to seller account

➤➤ Webhook Execution Flow (Post-Payment)

1. Mark track status = SOLD

   * Remove from public grid instantly

2. Trigger Stripe Connect payout split

3. Automated Legal Generation (PDFs)

   * Copyright Transfer Agreement
   * Non-Disclosure Agreement

   Must include:

   * Buyer Name/Entity
   * Seller Name/Entity
   * Track Title
   * Purchase Price
   * Date
   * Unique Transaction ID

4. Unlock S3 download links

5. Send email:

   * Invoice
   * Download links
